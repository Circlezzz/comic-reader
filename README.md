# comic reader
A comic reader for .cbz and .cbr format files. 

## features
- [x] night mode
- [x] fullscreen
- [x] .cbz support
- [x] .cbr support
- [ ] .cbt support

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
